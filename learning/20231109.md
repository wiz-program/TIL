# Rangeオブジェクトについて
Rangeオブジェクトを使用することで範囲指定をすることができる。
以下のように範囲指定をすることで、記述を大幅に削減することができる
```ruby
(1..5).each do |i|
 puts i
end
=>
1
2
3
4
5
```
以下のように配列で範囲指定することもできる
```ruby
alpha = ["a", "b", "c", "d", "e"]

p alpha.[](Range.new(2, 4)) #=> ["c", "d", "e"]
p alpha.[](2..4)            #=> ["c", "d", "e"]
```
## まとめ
Rangeオブジェクトは複雑なコーディングをする際、かなり役立つので積極的に使用していきたい
